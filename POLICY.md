Privacy Policy
for WhatsApp Guardian - Session Watchdog

Last updated: 2026-04-09

This Privacy Policy applies to the browser extension WhatsApp Guardian - Session Watchdog, developed by Thorsten Bylicki / BYLICKILABS.

---

1. General Information

Protecting personal data is an important priority. The extension WhatsApp Guardian - Session Watchdog has been developed to locally monitor, display, and technically log security-relevant states within WhatsApp Web sessions on the user’s device.

The extension is designed as a defensive monitoring tool for local browser session awareness.

---

2. Responsible Party

Thorsten Bylicki / BYLICKILABS
GitHub: https://github.com/bylickilabs

---

3. Purpose of the Extension

The extension is designed to locally evaluate certain technical states of WhatsApp Web within an active browser session, including in particular:

• detection of relevant session states
• detection of possible QR login states
• local event logging
• heartbeat monitoring of active sessions
• presentation of technical status information inside the extension UI

The extension is not intended for monitoring the content of private communications.

---

4. Data Processed

The extension processes only technically required information to the extent necessary for its functionality. This may include in particular:

• detected session modes such as active, unknown, or QR login
• page title information within WhatsApp Web
• unread state counts where technically derivable from the page title
• browser tab focus and visibility states
• local event logs stored by the extension
• user settings stored locally

---

5. No Collection of Message Content

The extension is not designed to read, store, analyze, or transmit private message contents, chat histories, contact lists, media files, or other communication content.

No content analysis of private communication is performed.

---

6. Local Storage

All data handled by the extension is processed and stored locally within the user’s browser storage to the extent required for the extension to function.

This includes in particular:

• configuration and settings
• local event logs
• session monitoring status values

The extension itself does not transmit this data to external servers.

---

7. No Sharing with Third Parties

By default, the extension does not send data to third parties, cloud services, analytics platforms, or advertising services.

No personal or session-related data is shared externally by the extension.

---

8. Extension Permissions

The extension only uses the permissions required for its technical functionality. These may include in particular:

• storage for local settings and event logs
• tabs for detecting relevant browser tabs
• notifications for local security-related alerts
• host permissions for WhatsApp Web where required for technical monitoring

These permissions are used solely for the purposes described in this Privacy Policy.

---

9. Export Functions

If the extension provides JSON or CSV export functions, export is performed only upon explicit user action. Exported data is generated locally by the user and is not automatically transmitted to third parties.

The user is solely responsible for any further handling of exported files.

---

10. Data Security

The extension has been developed with a security-oriented approach. However, the actual safety of the local environment also depends on factors outside the extension, including the browser used, the operating system, other installed extensions, and the overall security status of the device.

---

11. No Responsibility for Third-Party Environments

The extension operates in the context of browsers and third-party websites. No responsibility can be assumed for content, technical changes, or security mechanisms of WhatsApp Web, browsers, extension stores, or other third-party environments.

---

12. Changes to this Privacy Policy

This Privacy Policy may be updated if the extension’s functionality, technical framework, or legal requirements change.

The current version shall always apply.
